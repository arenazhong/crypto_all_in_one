# **什么是USDT？USDT如何运作？**

我们使用金钱的方式正在演变。作为传统法定货币数字化的先驱，世界上第一个稳定币Tether（泰达币）颠覆了传统金融系统。USDT兼具传统法定货币的稳定性和简易性，同时融入区块链技术，是币圈与真实法定货币的完美结合。USDT与现实世界的美元挂钩，可1:1兑换。USDT按以下五个步骤运作：

**步骤1：** 经过身份验证的Tether用户（可以是个人、交易所、商业公司或交易团队）将美元存入Tether的银行账户。

**步骤2：** Tether公司发行USDT代币，并将其发送到用户提供的钱包地址。市面上发行的USDT = 用户存入的美元 - 用户提现的美元。

**步骤3：** USDT用于用户交易，可转移、交易或储存以备后续使用。

**步骤4：** 用户可将USDT兑换成美元，只需将USDT存入其tether.to账户。

**步骤5：** Tether公司从市场上销毁对应数量的USDT，并将对应数量的美元转给用户。Tether公司是唯一可发行并销毁USDT的公司，以确保每颗USDT背后都有一美元作为保障。

除上述过程外，用户可在交易所购买USDT，或向现有USDT持有者购买。

# **购买虚拟货币（比特币BTC）**

购买的USDT会存放在资金账户中。点击首页右下角的【资产】，找到【资金账户】，即可查看已购买的USDT。若想用USDT交易其他虚拟货币，点击上方的【划转】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9802c95823_1729658328-64.jpg)

将资金从资金账户划转到交易账户，币种选择USDT，数量设为最大，点击【确认】。资金转至交易账户后，方可进行交易。点击下方中间的【交易】按钮。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/340ea144a0_1729659132-67.jpg)

在上方搜索框中输入想交易的币种，例如比特币（代码BTC）。选择“BTC/USDT”，即使用USDT兑换比特币。如需购买其他币种，也可在此搜索并用USDT兑换。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/faaa5fc78e_1729659934-70.jpg)

上方选择【买入】，然后选择下单方式，包括限价委托、市价委托和止盈止损等。限价委托和市价委托是最常用的两种方式。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9d67a77cda_1729660082-72.jpg)

“限价委托”是指你设定一个价格，只有当市场价格接近该价格时才能成交。例如，当前价格为66865，你想以66000的价格购买，则输入目标价格和购买数量或金额，点击【买入BTC】即可下单。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/20d2fc4941_1729660272-73.jpg)

下单成功并不代表成交，只是交易请求已提交。你可以在下方【委托】处查看挂单，需等待价格跌至目标价才能成交。若不想等待，可点击【撤单】取消购买请求。注意，挂单和撤单若不成交，不收取任何手续费。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/039afa6f11_1729660993-74.jpg)

若想立即购买，可选择“市价委托”。它会以当前市场最优惠的价格直接成交。输入购买金额，点击【买入BTC】即可成功购买。现在，你已经成功购买了比特币。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9c315ad3a0_1729661202-75.jpg)

购买比特币后，如何查看？点击右下角的【资产】，选择【交易账户】，即可看到已购买的比特币。
