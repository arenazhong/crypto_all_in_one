# 如何在OKEx绑定银行卡？

将银行卡绑定到您的OKEx（现为OKX）账户，可以便捷地进行法币充值和提现操作。以下是详细的操作步骤和注意事项。

# **绑定银行卡步骤**

**1）登录OKEx账户**

1. 打开OKEx App或访问OKEx官网，输入账户信息登录。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0fb3f6db68_1718090812-47.jpg) 

2. 如果您尚未拥有OKEx（欧易）账户，点击下方注册链接创建账户。

**欧易：[https://www.okx.com](https://www.okx.com/join/74873351)**

注册时填写邀请码：**74873351**，可节省20%手续费。按照提示完成注册后登录账户。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/99fbadcb93_1717402413-54.jpg) 

**2）进入买币页面**

在首页选择“C2C买币”，点击购买页面右下角的“我的”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ac6915a810_1718354682-50.jpg) 

**3）进入绑定页面**

1. 选择“收款账户管理”，点击“添加收款账号”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/278d974dfa_1718354849-53.jpg) 

2. 点击“收款方式”，选择“银行卡”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/876bac75a6_1718354998-56.jpg) 

**4）输入银行信息**

输入与账户注册信息一致的姓名、银行卡号和开户银行，点击【完成添加】。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ad06822185_1718355151-57.jpg) 

**5）完成绑定**

系统可能要求进行安全验证（如短信或谷歌验证码）。完成提示步骤后，绑定即成功。

# **注意事项**

**1）信息准确性**

  * 确保持卡人姓名与OKEx账户注册信息完全一致，否则绑定可能失败。
  * 银行卡及其他信息必须准确无误，以避免绑定问题或后续使用障碍。

**2）安全验证**

  * 绑定过程中可能触发双重验证（2FA），以保障账户安全。
  * 定期检查并更新账户安全设置，确保安全防护有效。

**3）银行支持**

确认OKEx支持您的银行。若不支持，需更换银行或选择其他支付方式。

# **常见问题**

**1）绑定失败怎么办？**

  * 检查银行卡信息是否正确，特别是卡号和姓名。
  * 部分银行可能限制绑定行为，请联系银行确认是否允许关联加密货币交易所。
  * 若多次尝试失败，建议联系OKEx客服寻求帮助。

**2）如何解绑或更换银行卡？**

  * 进入“银行卡管理”页面，选择需解绑的银行卡，点击“解绑”或“删除”。
  * 解绑原有银行卡后，按上述步骤绑定新卡。

# **结论**

遵循以上步骤，您可顺利在OKEx（OKX）绑定银行卡，实现便捷的法币充值和提现操作。
