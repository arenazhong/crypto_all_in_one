# 币安是哪个国家的？币安的老板是谁？币安是否安全？

币安创始人赵长鹏是江苏连云港人，后来随父母到了加拿大。赵长鹏是在加拿大长大的，是加拿大籍华人。最开始币安创立于上海，后来2017年9月4日之后，币安搬到了日本。经过多地的辗转。目前币安在意大利、台北、土耳其、伊斯坦布尔、迪拜都设有办公室。目前赵长鹏也是全球飞，主要在迪拜办公。对于币安这种国际化平台来说，讨论它是哪个国家的没有意义，老外们看币安就跟看TikTok是一样的。CEO是华人，就觉得是中国公司。实际上法律主体并不在中国。

币安是全球第一大交易所。占了币圈45%的交易量。接下来我就来为你介绍如何注册币安、下载币安以及在币安进行交易。

首先我们要下载币安的APP，安卓手机的用户直接扫描下方二维码来下载币安APP。扫描的时候如果手机自带的浏览器拦截你，就改用火狐浏览器。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/96d4cce376_2024020603140611.jpg) 

苹果手机的用户可以查看下方视频，去注册一个海外的Apple ID

然后打开appstore，点击右上角的头像，换登你的ID。换登完成之后，再搜索Binance，选择第二个进行下载就可以了。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/11311c1587_1707189435-1.jpg) 

还没有注册币安账户的朋友可以点击下方币安注册链接，去注册一个币安账户。

**<https://www.binance.com/zh-TC/join?ref=36457687>**

注册的时候可以直接使用你的中国大陆手机号，也就是+86的手机号去注册。填写邀请码36457687，就能够永久的获得20%的手续费优惠。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/d4b2ade684_1707189731-2.jpg) 

在币安交易之前，需要先去购买USDT。点击首页上方的【充值】，选择【快捷买币】

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/edd79adf7b_1707189989-3.jpg) 

在上方可以选择自选区或快捷区，这里我们选择“快捷区”。

右上角的“CNY”就是人民币的意思，就是我们要使用的法币。可以点击他进行切换你要使用的法币。这里我们就以CNY为例。

选择“买入”，币种选择“USDT”,USDT就是泰达币，目前币圈使用最广的就是USDT。USDT其实就类似于美元，当你使用你的法定货币购买了USDT之后，你就能够使用USDT来兑换，其他的任意几百种数字货币。

输入要购买的金额，点击“买入USDT”

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/cf8f622af7_1707190413-4.jpg) 

支付方式有支付宝、微信和银行借记卡3种。可以任意选择你习惯的支付方式。我们选择“银行借记卡”，他会告诉我们支付200人民币将会受到26.45颗USDT。点击“确认购买”

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/098c7199bf_1707190658-5.jpg) 

平台会自动为我们匹配好商家，根据商家给的提示，把钱转给商家。等待卖家放币给我们就可以了。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4e758ec6cb_1707190956-6.jpg) 

购买好的USDT都会在资金账户里面。点击右下角的【资金】上方选择【资金账户】，在这里就可以看到，我们购买好的USDT了。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/37466e16a4_1707191101-6.jpg) 

点击【划转】，从资金账户划转到现货账户，币种选择USDT，全部划转，点击“确认划转”。划转成功。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/f5b07f9ccc_1707191735-7.jpg) 

把USDT划转到现货账户之后，就可以进行交易了。

点击下方的【交易】按钮，进入交易页面。上方选择【现货】

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/3189324af0_1707192380-8.jpg) 

双箭头这里搜索我们要交易的币种，比如我们想要交易比特币，就搜索BTC，BTC就是比特币的代码,选择BTC/USDT。就是我们用USDT去兑换比特币。同样的，如果你想兑换其他的币种，都可以在这里进行搜索然后使用USDT去兑换。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/d7522dd4c9_1707192503-9.jpg) 

选择“买入”。这里有市价单、限价单还有止盈止损等。限价单和市价单是我们最常用的。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/bfcbab42ba_1707192980-10.jpg) 

限价单就是输入我们想要购买的金额，等卖家来跟我们成交。比如现在价格是37330，我们想以37320的价格来购买。在右侧输入37320，输入要购买的金额，点击“买入BTC”,下单成功。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/5e96104723_1707193431-11.jpg) 

这里所谓的下单成功并不代表成交。只不过是交易请求成功。可以在下方“当前委托”看到我们的单子就挂在下方。它需要跌到我们想要的目标价格，才能成交。如果你不想等了。可以点击后面的“撤单”把购买撤销掉。点击“撤单”订单已撤销。挂单以及撤单如果不成交，是不收取任何手续费的。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/6843abfcf4_1707194182-14.jpg) 

如果我现在就想买，就选择市价单。市价单就是它会以当前市场最合适的价格给我们。直接输入要购买的金额就可以购买成功。输入购买的金额，点击“买入BTC”，下单成功。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/9046f5bf55_1707194627-17.jpg) 

现在比特币就购买成功了，点击右下角的【资金】，上方选择【现货账户】，就可以看到我们购买好的比特币了。
